Requirements

  • ESP32 development board (ESP32-WROOM recommended)
  • USB cable for flashing
  • Addressable LED strip (WS2812B, SK6812, etc.)
  • Bambu Lab printer (X1, X1C, P1P, or P1S)
  • Chrome, Edge, or Opera browser on desktop

After Installation

  1. Configure WiFi: After flashing, you'll be prompted to connect to your WiFi network directly in this browser
  2. Access the Web UI: Once connected, click the link provided to open the device's configuration page
  3. Configure Printer: Enter your Bambu Lab printer's IP address, serial number, and access code
  4. Set up LEDs: Configure your LED strip type and color preferences

Alternatively, if WiFi setup doesn't appear, the ESP32 will create a WiFi access point named BLFLC_AP. Connect to it and navigate to 192.168.4.1.
